## Project Description
This Android application helps users convert essential travel-related values such as currency, fuel efficiency, and temperature.

## Features
- Currency Conversion (USD to AUD, EUR, GBP, JPY)
- Fuel Conversion (mpg to km/L)
- Temperature Conversion (Celsius, Fahrenheit, Kelvin)

## Technologies Used
- Java
- Android Studio
- XML Layout

## How It Works
The app uses dropdown menus (Spinners) to select conversion types and units. Users input a value and press the convert button to get results instantly.
